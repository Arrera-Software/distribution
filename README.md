# Arrera Distribution

[Read in English](README_en.md)

Depots qui sert a stocker les fichier de depots pour la distribution des application Arrera par l'intermédiaire d'Arrera Hub

## Explication 

Ce dépôt a pour fonction principale d'héberger les fichiers de configuration (fichiers de dépôts) nécessaires au fonctionnement d'**Arrera Hub**.

Il agit comme un catalogue centralisé qui permet à Arrera Hub de :
*   Répertorier les applications disponibles au téléchargement.
*   Vérifier la disponibilité de nouvelles mises à jour.
*   Récupérer les informations de versionnage et les liens de téléchargement des applications de l'écosystème Arrera.

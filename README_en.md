# Arrera Distribution

Repository used to store repository files for the distribution of Arrera applications via Arrera Hub.

## Explanation

The main function of this repository is to host the configuration files (repository files) necessary for the operation of **Arrera Hub**.

It acts as a centralized catalog that allows Arrera Hub to:
*   List applications available for download.
*   Check for the availability of new updates.
*   Retrieve versioning information and download links for applications in the Arrera ecosystem.
